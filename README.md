# 🔍 Visualizador de Perfil do GitHub

Este é um projeto prático desenvolvido para buscar e exibir informações de perfis do GitHub em tempo real, utilizando a API oficial do GitHub.

## 🚀 Funcionalidades

- **Busca em Tempo Real:** Digite o nome de usuário e obtenha os dados instantaneamente.
- **Informações do Perfil:** Exibe foto de avatar, nome, biografia e link para o perfil.
- **Estatísticas de Engajamento:** Mostra o número de seguidores e quantas pessoas o usuário segue.
- **Lista de Repositórios:** Exibe os 10 repositórios mais recentes com detalhes como:
  - ⭐ Quantidade de Estrelas
  - 🍴 Quantidade de Forks
  - 👀 Observadores (Watchers)
  - 💻 Linguagem principal utilizada
- **Design Responsivo:** Adaptado para diferentes tamanhos de tela (Celular, Tablet e Desktop).
- **Animações Fluidas:** Interface interativa com transições suaves.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação da página.
- **CSS3:** Estilização, animações e responsividade.
- **JavaScript (ES6+):** Lógica de busca, consumo da API e manipulação do DOM.
- **GitHub API:** Fonte dos dados dos usuários.
- **Devicon:** Ícones estilizados de tecnologia.

## 💻 Como Executar o Projeto

Como este é um projeto puramente de front-end (HTML, CSS e JS), você não precisa instalar nada!

1. Faça o download ou clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/visualizador-perfil-github.git
   ```
2. Entre na pasta do projeto.
3. Abra o arquivo `index.html` no seu navegador preferido.

> **Dica de Iniciante:** Se você usa o VS Code, recomendo a extensão **Live Server**. Basta clicar com o botão direito no `index.html` e selecionar "Open with Live Server" para ver as alterações em tempo real enquanto você estuda o código.
